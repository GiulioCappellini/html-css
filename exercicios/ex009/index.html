listas

Listas ordenadas:

Listas ordenadas são listas que tem uma ordem para ser seguida sua tag é <ol> essas listas podem ser por ordem numérica(1), alfabetica com letras maiusculas(A), alfabetica com letras minusculas(a), com numeração romana maiuscula ou com numeração romana minuscula. o parâmetro dentro de <ol> que deve ser passado para definir este tipo é 'type='exemplo'' e para definir, por exemplo em qual número a lista vai começar é utilizado o parâmetro 'start='exemplo'', além de tudo isso cada item da lista deve ser adicionado com a tag <li>. Abaixo tem um exemplo de uma lista ordenada com números romanos e começando do número 5:

  1. Acordar
  2. Tomar banho
  3. Tomar café da manhã
  4. Se arumar
  5. Passar desodorante e etc
  6. Ir para a escola

Listas não ordenadas

Listas não ordenadas são listas que não posuem uma ordem expecifica a ser seguida como por exemplo uma lista de super mercado, sua tag é <ul> e o único parâmetro que ela contem é 'type='exemplo'' que pode ser uma bolinha prenchida(disc), uma bolinha vazia(circle) ou um quadrado(square), assim como nas listas ordenadas cada item deve ser adicionado utilizando a tag <li>. Abaixo tem um exemplo de uma lista não ordenada com uma bolinha vazia:

  • Acordar
  • Tomar banho
  • Tomar café da manhã
  • Se arumar
  • Passar desodorante e etc
  • Ir para a escola

Misturar listas

É importante saber que além de ser possivel colocar listas dentro de outras listas, também é possivel misturar listas ordenadas com listas não ordenadas com a ajuda do aninhamento no código. Abaixo tem um exemplo de uma lista da série escolar e minhas matérias escolares favoritas e os assuntos preferidos dessa matéria:

    1. Educação física

    2. Matematica

      2. divisão
  • 6oano ao 9oano

      1. Só as aulas pratica, independentemente do jogo
  • 1omédio até hoje
    1. Física
      1. Perguntas sobre inercia
      2. Todo assunto que envolvesse velicidade e tipos de energia

Listas de definição

As listas de definição se acemelham muita a um dicionário onde em cima e colado a esquerda se tem o termo e logo em baixo com uma pequena margem se tem a definição, em HTML5 existem tags para isso, a tag <dl> é usada para uma lista de definição, a tag <dt> indica o terma a ser definido e a tag <dd> é a definição deste termo, neste caso a tag <li> não será utilizada já que o conteudo será adicionado com as tags fornecidas anteriormente. Abaixo está um exemplo de uma lista de definição das linguagens para desenvolvimento web:

HTML
Utilizada para fazer o esqueleto do site(linguagem de marcação)
CSS
Utilizada para dar estilo e personalizar o site(linguagem de forma)
JavaScript
Usada para adicionar as interações com o usuário e as animações do site(linguagem de programação)
