tag <code>

Essa tag faz com que todos entendam que seu conteudo se trata de um código, por isso seu nome é esse. Abaixo tem um exemplo de um código, em python, com a tag <code>:
número = int(input('Digite um número qualquer: '))

tag <pre>

Essa tag alem de tornar seu conteudo monoespaçado faz com que todos os espaços e quebras de linhas sejam considerados e mostrados no programa final. Abaixo tem um código, em python, que foi escrito da mesma forma que você verá graças a essa tag, por facilitar a vida do desenvolvesor em html5 ela se torna muito util! 


n = int(input('Digite um número qualquer: '))
if n % 2 == 0:
    print('Seu número é PAR')
else:
    print('Seu número é IMPAR')

tag <q>

Essa tag apênas substitui as aspas, mas é melhor usar essa tag no lugar das aspas sempre que o conteúdo for uma citação simples já que essa tag é semântica.

tag <bq> / <blockquote>

Essa tag tem a mesma função que a de cima, abaixo você pode-rá ver a diferença entre elas:
com <q>:
Um velho uma vez disse O computador é apênas um burro muito rapido
com <bq>:
um velho uma vez disse

O computador é apênas um burro inteligente!

tag <abbr>

Essa tag tem a função de mostrar ao usuário oque a abreviação ou sigla significa, atraves de um pequeno pop-up que se abre ao lado da abreviação quando o usuário passa o cursor em cima da sigla. Abaixo terá um exemplo:
pqp

tag <bdo:r>

Essa tag tem a função de simplesmente inverter seu conteudo, como mostrado no exemplo a seguir:
Estou adorando aprender HTML5 e CSS3, não vejo a hora de começar a aprender JavaScript
na verdade não é que ela inverte o conteudo, é que ela o escreve da direita para a esquerda.
