Formatação de textos

Negrito / Strong

Existem 2 formas de deixar o texto em negrito:

Não semântica, utilizando a tag < b >: ❌
Exemplo de frase em negrito

Semântica, utilizando a tag < strong >: ✅
Exemplo de frase em negrito

Italico / Ênfase

Não semantica, utilizando a tag < i >: ❌
Exemplo de frase em italico

Semantica, utilizando a tag < em >: ✅
Exemplo de frase em italico

Conclusão

É sempre melhor utilizar uma tag semantica no lugar de uma que não é semantica, oque muda é que para quem vai ler o código fará mais sentido e não corre o risco de deixar de funcionar em pouco tempo!

Outras tags de formatação:

Texto marcado

Utilizando a tag < mark > é possivel criar uma marcação no texto e é possivel trocar a cor da marcação, caso queira trocar apenas a cor de um < mark > basta adicionar 'style="background-color:cor desejada"', caso queira mudar a cor de todos os < mark > do programa a alteração deve ser feita dentro de head, do mesmo jeito que tricamos o bg(background-color) da pagina em si, basta mudar o termo 'budy' por 'mark'.

Texto grande e pequeno

Também é possivel mudar o tamanho da letra com html5, mas isso não é muito semântico então é melhor fazer com css, mas a tag(< small >) para texto pequeno pode ser semântico e mesmo assim seu resultado não é tão significante.

Texto deletado

É possivel criar um texto deletado com a tag < del > o texto deletado nada mais é doque um texto que deve ser lido mas não deve ser considerado.

Texto inserido

Um texto inserido nada mais é doque um tecto sublinhado que pode ser feito da forma semântica e da forma errada(não Semântica):
Forma semântica: Este é um exemplo de texto sublinhado.
Forma errada(não semântica):Este é um exemplo de texto sublinhado.
A forma semântica usa a tag < ins > já a forma errada usa a tag < u >

Texto suberscrito

Um texto suberscrito é aquela letra ou número que está um pouco acima e menor que o normal, ela é feita com a tag < sup > e torna possivel escrever algumas contas matematicas como o teorema de pitagoras que é a2 = b2 + c2.

Texto subcrito

Um texto subcrito e a mesma coisa que o texto subrescrito porem ao contrario, inves da letra ou número ser em cima ele será em baixo como na formula da água: H2O
