Alguns truques sobre linhas e paragráfos:

Para adicionar a tag '<' é necesário digitar '&, l, t, ;' e para '>' basta digitar '&, g, t, ;'

Para fazer a linha acima basta utilizar a tag '<hr>'.

Para dar o espaçamento de 1 linha é utilizada a tag '<br>' como no exemplo a seguir:
bom
dia
como
você
está
?

Para adicionar um comentario basta digitar '<!--' e dar enter, o VScode irá completar e já será possivel degitar o comentario que não apareserá no site, obviamente.

Agora para adicionar simbolos como dolar, libra, delta e etc existem comandos especificos: ¢ £ Δ, sertamente não é possivel decorar todos estão sempre que for necesário basta pesquisar na web.

Para adicionar emojis é necesário um código que pode ser adquirido no site 'emojipidea' como 🤙 e 🤘 utilizando o código '&#x' seguido do código, sem o 'u+', do emoji e finalizando com ';'.
