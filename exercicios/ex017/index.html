Este parâmetro define a fonte do texto, assim como na tag <source:media> existe uma sequencia "tente este primeiro, se não funcionar tente esse, se nenhum deles funcionar utilize esse" a diferença é que neste parâmetro, a ultima é sempre uma genérica como "serifa, sans-serifa, monospace" esse ultimo manda o código encontrar qualquer fonte que faça parte deste grupo.
